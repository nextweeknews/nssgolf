Overall GPI

Overall GPI uses the Ranked League rating as the baseline, then applies a reliability-weighted tournament adjustment after translating tournament performance onto the Ranked League scale.

  • Eligibility if tournament_matches < 3: total_gpi = ranked_rating
  • Standardization ranked_z = (ranked_rating - ranked_mean) / ranked_std tournament_z = (tournament_rating - tournament_mean) / tournament_std tournament_delta_z = tournament_z - ranked_z
  • Reliability tournament_reliability = min(1, tournament_weighted_matches / 20)
  • Adjustment if tournament_delta_z >= 0: tournament_delta = reliability * 120 * tanh(tournament_delta_z / 1.5) if tournament_delta_z < 0: tournament_delta = reliability * 60 * tanh(tournament_delta_z / 1.5)
  • Total total_gpi = ranked_rating + tournament_delta

Ranked League Rating

Ranked League GPI is an opponent-aware composite built from the full ranked sample, peak performance, and recent player-aware form.

  • Composite ranked_rating = 0.50 * full_history_pl + 0.25 * peak_pairwise_elo + 0.25 * recent_form_pl
  • Full History full_history_pl = opponent-aware weighted pairwise Plackett-Luce over all tracked Ranked League matches
  • Potential peak_pairwise_elo = highest weighted pairwise Elo reached during the ranked replay
  • Recent Form recent_form_pl = opponent-aware weighted pairwise Plackett-Luce from each player's last 100 Ranked League matches
  • Lobby Weight match_weight = min(3, 1 + 0.7 * log2(player_count - 1)) pair_weight = match_weight / (player_count - 1)

Tournament Rating

Tournament GPI is independent from the Ranked League model. It uses flat Plackett-Luce ratings from solo 1v1 tournament results, with each tournament match counted as twice the weight of a Ranked League match.

  • Model tournament_rating = flat Plackett-Luce rating from tracked solo 1v1 tournament matches
  • Match Weight tournament_weighted_matches = tournament_matches * 2
  • Reliability tournament_reliability = min(1, tournament_weighted_matches / 20)
  • Recency recency_weight = none
